1. 사용자 인증/인가

Spring Security 폼 로그인과 세션 기반 인가

로그인/로그아웃

  • 로그인: 이메일(=username) + 비밀번호, 폼 경로: /member/signin.
  • 성공 시 기본 리다이렉트: /, 실패 시: /member/signin?error.
  • Remember‑Me 사용 시 14일 쿠키 발급.
  • 로그아웃: /member/signout → 세션 무효화 + Remember‑Me 삭제 → /?logout.

권한/인가

  • 역할: ROLE_USER, ROLE_ADMIN.
  • 글/스토리/공지 작성·수정·삭제 및 일부 관리 기능은 관리자만.
  • 목록/상세 조회와 정적 리소스 접근은 익명 허용 경로를 명시.

비밀번호 저장

  • 개발: NoOpPasswordEncoder(평문) — 운영 전환 시 반드시 해시(BCrypt 등)로 교체.
