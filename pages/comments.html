4. 댓글

CRUD, 작성자 검증, 좋아요(AJAX)

기능

  • 생성: POST /posts/{postId}/comments (인증) — 작성자는 로그인 사용자명, 완료 후 상세 리다이렉트.
  • 수정: POST /comments/{commentId}/update (인증) — 본인 검증 실패 시 ?error=... 부여.
  • 삭제: POST /comments/{commentId}/delete (인증).
  • 댓글 좋아요: POST /comments/{commentId}/like (인증, AJAX) — 성공 시 카운트/텍스트 토글, 401 시 로그인 안내.
  • 메시지 표시: 상세 상단에 param.success/param.error 알림 배너.
