5. 좋아요

게시글/댓글 좋아요 토글과 401 처리

엔드포인트

  • 게시글: POST /posts/{postId}/like (인증) → { isLiked, likeCount } 반환.
  • 댓글: POST /comments/{commentId}/like (인증) → 카운트/텍스트 토글.
  • 인증 실패: 401 + {"message":"UNAUTHORIZED"}.
