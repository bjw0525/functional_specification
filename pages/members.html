2. 회원 관리

가입/수정/탈퇴와 관리자 기능

회원가입

  • POST /member/signup에서 username(이메일), password, nickname, email, phoneNumber, address 저장.

회원정보 조회/수정

  • 인증 사용자만 /member/update 접근.
  • 현재 비밀번호 검증 후 이메일/연락처/주소 변경, 비밀번호는 입력 시에만 변경.

회원탈퇴

  • POST /member/delete 시 계정 삭제 → 세션 무효화/Remember‑Me 삭제 → /?accountDeleted.

관리자 기능

  • 태그 관리 /member/tag-update, 소개글 관리 /member/introduction-update — 관리자만.
