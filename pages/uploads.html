8. 파일 업로드 API

이미지 업로드와 임시 파일 정리

엔드포인트

  • 이미지 업로드: POST /api/uploads/image (multipart 키 file, 최대 4MB) → {"url":"/uploads/images/..."}.
  • 임시 정리: POST /api/uploads/cleanup 에 usedImages 전달 → 미사용 이미지 삭제, {"deleted": n} 반환.
  • 경로: 서버 로컬 uploads/images, URL: /uploads/images/....
